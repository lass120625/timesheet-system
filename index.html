📊 工時表格式說明

Payroll ID, First Name, Last Name, Start Date, Work Week, Hours, Level, Rate, Farm EQ-0001, John, Doe, 2024-01-15, 2024-01-15, 45, Senior, 25.50, Farm A EQ-0001, John, Doe, 2024-01-15, 2024-01-22, 38, Senior, 25.50, Farm A EQ-0002, Jane, Smith, 2024-02-01, 2024-02-05, 42, Junior, 20.00, Farm B

重要格式說明：

  • 📅 Start Date：員工在該農場工作的第一天，格式為 YYYY-MM-DD 或 DD/MM/YYYY
  • 📅 Work Week：工作週的週一日期，格式為 YYYY-MM-DD 或 DD/MM/YYYY
  • 🔢 週數計算：系統會自動根據 Start Date 和 Work Week 計算正確的週數
  • 🏭 Farm/Level/Rate 更新：如果員工轉換農場或調整薪資，新資料會覆蓋舊資料

上傳後自動處理：

  • ✅ 驗證 EQ-0001 格式的員工編號
  • ✅ 自動計算正確的工作週數
  • ✅ 支援農場和職級異動
  • ✅ 整合新舊資料，檢測重複工作週
  • ✅ 自動儲存到瀏覽器本地
